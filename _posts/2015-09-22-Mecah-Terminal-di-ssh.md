Pernah denger tmux? pernah lah pasti. pake itu asik lho.

``` go
$ sudo apt-get install tmux
```

untuk menjalankanya tinggal jalanin comand

``` go
$ tmux
```
satu window tmux bisa terdapat beberapa panel dan tmux bisa buat window baru pula.
berikut beberapa comand yang sering digunakan

- *Ctrl+b (lepas) + "* - split panel horisontal.
- *Ctrl+b (lepas) + %* - split panel vertikal.
- *Ctrl+b (lepas) + tombol panah (arrow key)* - pindah panel.
- *Ctrl+b (tahan) + tombol panah (arrow key)* - resize panel.
- *Ctrl+b (lepas) + c* - (c)reate , Buat Window Baru.
- *Ctrl+b (lepas) + n* pindah ke window berikutnya (n) = (next).
- *Ctrl+b (lepas) + p* pindah ke window sebelumnya (p) = (previous).
