Kemaren kan udah tu install ruby. sekarang aku mau install [jekyll](http://jekyllrb.com/) ni.
Jekyll itu kayak *blog engine* gitu tapi nulisnya pake markdown. dan templetingnya enak.
coba deh habis baca ini install.

pertama aku buka *terminal* abis itu jalanin command

``` python
~ $ gem install jekyll
~ $ jekyl new nama_blog_kamu
~ $ cd nama_blog_kamu
~/nama_blog_kamu $ jekyll serve
```

dah gitu aja.
abis itu aku buka di browser `localhost:4000`
coba deh kalau gak percaya.
